## Installation

```bash
$ npm install
```

## Setup

```bash
$ cp .env.example .env
```

> You can define consts in `src/config/app.config.ts` and export them from there.

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
